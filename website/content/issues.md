# Report Issues

Found something that's not working right? Let us know!  You can report bugs directly on GitHub Issues or reach out [via the contact page](/contact).

## GitHub Issues

The best way to report bugs or request features is through our GitHub repository:

[Link to GitHub](https://github.com/ghostops/LiveOL/issues)

## Bugs

Bugs are handled as soon as possible, but LiveOL is a one-person hobby project, so no guarantees on timelines. 

**LiveOL+ subscribers will be prioritized.**

## What to Include

When reporting an issue, please provide as many of these as possible:

1. **Description**: Clear explanation of the problem or request
2. **Steps to Reproduce**: How can we recreate the issue?
3. **Expected Behavior**: What should happen?
4. **Actual Behavior**: What actually happens?
5. **Device/Browser**: What device and browser are you using?
6. **Screenshots**: If applicable, include screenshots
